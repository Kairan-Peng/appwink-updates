AppWink 0.1.21 Release Notes

AppWink v0.1.21

This release fixes two Hidden Bar regressions from 0.1.20: the expanded left-boundary marker can disappear against a black menu bar, and collapsing the expanded bar can feel delayed because the status-item pusher jumps to an oversized width. It keeps existing hotkey bindings, license state, update settings, and Toolbox preferences compatible with 0.1.20.
