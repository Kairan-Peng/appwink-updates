AppWink 0.1.23 Release Notes

AppWink v0.1.23

This release ships the Settings window redesign and the related menu, window, and polish follow-ups from the current worktree. It keeps existing hotkey bindings, shortcut assignments, direct-license state, Sparkle update settings, and feature-toggle values compatible with 0.1.22.
