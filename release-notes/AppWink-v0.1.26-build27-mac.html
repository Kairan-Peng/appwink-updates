AppWink 0.1.26 Release Notes

AppWink v0.1.26

This release fixes two Settings-window regressions: returning from Help now keeps the previous Settings scroll position, and AppWink's main/configuration and Settings windows can be brought to the front correctly from Mission Control or a three-finger swipe up. It keeps existing hotkey bindings, shortcut assignments, direct-license state, Sparkle update settings, and feature-toggle values compatible with 0.1.25.
