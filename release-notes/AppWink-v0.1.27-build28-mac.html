AppWink 0.1.27 Release Notes

AppWink v0.1.27

This release polishes the hotkey setup and Settings help experience. It shortens Settings feature names, moves detailed explanations into About/Help, adds an in-place hotkey guide on the hotkey configuration page, and improves hotkey-list reordering so dragging is easier to start from the left gutter while remaining stable at row boundaries. Existing app bindings, shortcuts, direct-license state, Sparkle update settings, and feature-toggle values remain compatible with 0.1.26.
