AppWink 0.1.28 Release Notes

AppWink v0.1.28

This release fixes a licensing enforcement gap in Settings Feature Toggles. Inactive license states can no longer enable or run the six Settings-controlled utility features. Existing app bindings, shortcuts, direct-license state, Sparkle update settings, and saved feature-toggle preferences remain compatible with 0.1.27.
